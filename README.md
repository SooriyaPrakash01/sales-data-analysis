# 📊 Sales Data Analysis

This project performs sales revenue analysis using Python.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib

## 📈 What This Project Does
- Calculates total revenue by product
- Calculates monthly revenue
- Visualizes product performance

## 💡 Key Insights
- Laptop generated the highest total revenue.
- February showed higher revenue growth compared to January.
